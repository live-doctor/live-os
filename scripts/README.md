# Homeio Scripts

This directory contains utility scripts for installing and maintaining Homeio.

## Install & Update

- `install.sh`: fresh install from a git branch.
- `setup.sh`: install system dependencies needed by Homeio.
- `update.sh`: update an existing Homeio installation.
- `uninstall.sh`: remove an installed Homeio instance.

## Repository Prep

- `prepare.mjs`: run repo-level setup steps required by npm scripts.
